## About Jujomi

This was developed as our final assignment at [SENA](https://en.wikipedia.org/wiki/National_Training_Service_(Colombia)), where we studied Software Development back in year 2022. The main objective with this was to apply all, or most of what we learned by creating a software solution.

Jujomi is a little web that allows you to create online courses about a wide variety of topics and share them with the world, featuring:
- **Course Management:** create, edit, erase, publish, unpublish and share courses/lessons.
- **User Account and Profile Management:** singup & login, see, update and erase info (email, profile pic, bio, username, password).

This project was inspired by e-learning platforms such as [FreeCodeCamp](https://www.freecodecamp.org/), [Platzi](https://platzi.com/) and [Udemy](https://www.udemy.com/).
